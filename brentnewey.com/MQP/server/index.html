Adaptive Websites Using Genetic Algorithms



This project is being undertaken as a Major Qualifying Project in the Computer Science Department at WPI.

Duration: A'02 - C'03. Now D'03

Problem Statement:

To find the optimal layout for a given hypermedia. That is, to find a solution that maximizes user efficiency. This will be accomplished using a self-modifying system based on a user model collected from usage statistics. These statistics will be processed using an algorithm that simulates evolutionary growth to obtain a solution starting from a random set.
